Survival Prediction:Titanic Analysis using KNN and LR

The Survival Prediction project of the Titanic is a data-driven analysis that aims to predict the likelihood of survival for passengers aboard the ill-fated RMS Titanic. By analyzing various factors such as age, gender, class, and family size, the project utilizes machine learning algorithms to build a predictive model. The ultimate goal is to create an accurate prediction model that can help understand the factors that influenced survival rates during this historic maritime tragedy.
